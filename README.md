# OUSSAMA HATTAN
## Cyber-Minimalist Engineering Portfolio V2

A high-fidelity digital presence engineered for the 2026 web. This portfolio focuses on architectural precision, motion-driven storytelling, and a premium dark-mode aesthetic. It serves as a comprehensive showcase of full-stack engineering capabilities, featuring advanced UI/UX patterns and multi-language support.

---

## Design Philosophy

The project adheres to a Cyber-Minimalist aesthetic, characterized by:

- Glassmorphism: Subtle translucent surfaces with blurred backgrounds.
- Kinetic Typography: Purposeful motion that guides the viewer's attention.
- High Contrast Lighting: Moody atmospheres punctuated by surgical-grade cyan and purple accents.
- Bento Grid Layouts: Structured information hierarchy inspired by modern dashboard aesthetics.

## Getting Started

### Prerequisites

- Node.js 18.0.0 or higher
- npm or yarn package manager

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/H-Ossama/Portfolio_V2.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure environment variables:
   Create a `.env.local` file in the root directory for contact form functionality.

4. Execute development server:
   ```bash
   npm run dev
   ```

---

## Localization Support

The portfolio supports three primary languages. To update content, modify the corresponding configuration file:

- English: `src/config/portfolio.en.ts`
- French: `src/config/portfolio.fr.ts`
- German: `src/config/portfolio.de.ts`
